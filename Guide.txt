1. Клонируем проект:
    > git clone https://github.com/ltropin/sport-center.git
2. Открываем проект и настраиваем строку подключения к базе данных в файле `appsettings.json`. (Должна быть создана пустая база данных!)
3. Далее мигрируем таблицу из проекта в базу данных. Открываем `Консоль диспетчера пакетов` находится в `Вид -> Другие окна -> Консоль диспетчера пакетов`. И пишем следующее:
    PM> Update-Database
4. После этой команды у вас создастся схема БД.
5. Нужно скопировать и вставить метод из файла `Hash.txt` в модель `Models/Client.cs`.
6. Запустить первый раз проект. Он заполнить тестовыми данными в БД.
